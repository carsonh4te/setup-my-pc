Usage:
	setuphelper [ACTION]
	setuphelper [ACTION] [OPTION]

Action:
	install 	Install the software related to the [OPTION] chosen
	version 	Show last version of setuphelper
	help 		Show this help message

Option:
	basic 		Install basic tools
	personal 	Install carsonh4te personal tools
	dev 		Install development tools

Example:
	setuphelper install personal
